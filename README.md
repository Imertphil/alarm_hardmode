# AlarmHardMode

It is time to get up. Procrastination prohibited. XD

## Overview

AlarmHardMode is a mobile application built for individuals who struggle with waking up, specifically designed with Neurodivergent (ADHD and ASD) users in mind. 

The standard practice of setting multiple alarms an hour before the actual wake time fragments sleep and reinforces sleep inertia. This application enforces a strict single alarm system. To silence the alarm, the user must complete active cognitive tasks. By forcing the brain into an active state through problem solving or memory recall, the application eliminates the possibility of falling back asleep.

## Core Mechanism

Traditional alarms rely on a simple physical button press. For an ADHD or ASD brain, this action requires zero cognitive effort, leading to an immediate sleep relapse. AlarmHardMode bridges the gap between physical arousal and mental wakefulness.

* Cognitive Wake Protocol: The alarm can only be dismissed by successfully completing a predetermined number of cognitive challenges.
* Math Operations: Users must solve arithmetic equations.
* Vocabulary Recall: Users must interact with specific vocabulary definitions.
* Absolute Engagement: The interface prevents passive swiping or mindless snoozing.

## Development

Built with React Native and Expo. The architecture strictly utilizes JavaScript and functional components for optimal performance and cross platform compatibility.
